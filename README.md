**BLOGAPP**

**Steps to start the project: -**

1. Clone the project in your local machine
2. `cd chat-app`
3. **Install dependencies**

   a. `cd backend/` <br />
   b. `npm install` <br />
   c. `cd frontend/` <br />
   d. `npm install` <br />

5. **Start the server**

   a. `cd backend` <br />
   b. `npm run dev` <br />

6. **Start the Client**

   a. `cd frontend` <br />
   b. `npm run dev` <br />
   <br />
